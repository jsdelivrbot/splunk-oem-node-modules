Packagecom.splunk.services.search
Classpublic class SearchJob
InheritanceSearchJob flash.events.EventDispatcher

The SearchJob class is used to dispatch a new search job or connect to an existing search job on a particular Splunk server.

refreshIntervalproperty 
refreshInterval:Number  [read-write]

The length of time, in seconds, between automatic refreshes of job properties. If set to zero or less, automatic refresh is disabled.

The default value is 0.25.
