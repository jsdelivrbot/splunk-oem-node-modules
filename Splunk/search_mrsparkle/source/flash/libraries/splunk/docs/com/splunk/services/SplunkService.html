Packagecom.splunk.services
Classpublic class SplunkService
InheritanceSplunkService flash.events.EventDispatcher

The SplunkService class connects to a Splunkd server. The default assumption is that this class is connecting to the Splunk UI server, and not directly to the Splunkd engine.

Method detail
authenticate()method
public function authenticate(username:String, password:String):void

Authenticates with the Splunk server specified in the hostPath property, using login credentials. If a service is already connected, the connection is closed.

Parameters
username:String — The username of a Splunk user.
 
password:String — The password associated with the username.

Throws
— The value of the username or password parameter is null.

loadSession()method 
public function loadSession(sessionKey:String, validate:Boolean = true):void

Connects to an existing authenticated session. This method is used if there is a managing service that has already obtained a session key. If a service is already connected, the connection is closed.

Parameters
sessionKey:String — The authentication token to use.
 
validate:Boolean (default = true) — Whether to validate the sessionKey by sending a request to the server.
