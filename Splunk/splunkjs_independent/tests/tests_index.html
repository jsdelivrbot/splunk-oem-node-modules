Login Tests

Run these tests to test login! and onSessionExpired.

  • test_login

    This page tests passing a dict to authenticate. It:

    • Checks that login succeeds
    • Checks that a search manager can get data

  • test_login_function

    This page tests pasing a function to authenticate. It:

    • Checks that login succeeds
    • Checks that a search manager can get data
    • Checks that the provided auth was called by looking for a marker placed by authenticate

  • test_reauth

    This page tests that the default reauthentication happens when the session key is invalidated. It:

    • Checks that login succeeds
    • Checks that a search manager can get data
    • Checks that a new login happens if the sessionKey was invalidated
    • Checks that a search manager can get data after the new login

  • test_reauth_function

    This page tests that a provided onSessionExpired function is called when the session key is invalidated.

    • Checks that login succeeds
    • Checks that a search manager can get data
    • Checks that a new login happens if the sessionKey was invalidated
    • Checks that a search manager can get data after the new login
    • Checks the marker left by the onSessionExpired function is present

  • test_username

    This page tests that the username returned by the authenticate function is set as the value for the config username

    • Checks that login succeeds
    • Checks that username is set

  • test login re-direction

    A page that automatically redirects to a custom login page if unauthenticated.

    • On clicking this page, you should be redirected to a login page
